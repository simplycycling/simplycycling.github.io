Essential Keyboard Shortcuts

This is the year I up my keyboard game.

Upper case indicates Shift is included.

Terminal shortcuts

Key combination Action
Ctrl + a Move to the beginning of the line
Ctrl + e Move to the end of the line
Ctrl + [left arrow] Move one word backward (on some systems this is Alt + B)
Ctrl + [right arrow] Move one word forward (on some systems this is Alt + F)
Ctrl + u (bash) Clear the characters on the line before the current cursor position
Ctrl + u (zsh) If you're using the zsh, this will clear the entire line
Ctrl + k Clear the characters on the line after the current cursor position
Esc + [backspace] Delete the word before the cursor
Ctrl + w Delete the word before the cursor
Ctrl + r Search history
Ctrl + g Escape from search mode
Ctrl + _ Undo the last change
Ctrl + l Clear screen
Ctrl + s Stop output to screen
Ctrl + q Re-enable screen output
Ctrl + c Terminate/kill current foreground process
Ctrl + z Suspend/stop current foreground process

Unity shortcuts

Key combination Action
Ctrl + Alt + j move window to next monitor (requires window jumper script)
Alt+F4 Closes window
PrtSc Takes snapshot of whole screen
Alt+PrtSc Take a screenshot of the current window
Shift+PrtSc Takes snapshot of selected area
Ctrl + Alt + T Open new terminal window

Window positioning

Key combination Action
Ctrl + Alt + 9 top right (25%)
Ctrl + Alt + 8 top half screen (50%)
Ctrl + Alt + 7 top left (25%)
Ctrl + Alt + 6 right half screen (50%)
Ctrl + Alt + 5 (100%)
Ctrl + Alt + 4 left half screen (50%)
Ctrl + Alt + 3 bottom right (25%)
Ctrl + Alt + 2 bottom half screen (50%)
Ctrl + Alt + 1 bottom left (25%)
Ctrl + Alt + 0 minimized

Chrome specific

Key combination Action
Ctrl + t Open new tab
Ctrl + w Close tab
Ctrl + Shift + T Re-open last closed tab
Ctrl + tab View next tab
Ctrl + 9 View previous tab
Ctrl + l Put cursor in search bar
Ctrl + backspace Erase current search
Ctrl + right arrow Highlight next word in search
Ctrl + left arrow Highlight previous word in search
Backspace Go back to previous page
Shift + Backspace Go forward to next page
